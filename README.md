scala-2.10.0-trial
==================

Scala 2.10.0で追加された機能を試したものです。

単純に「.scala」ファイルを配置しているだけでsbtなどは使用していないので、
```sh
$ scalac XYZ.scala
```
などのようにコンパイルしてください。
